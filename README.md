# Django Unit Converter

**Repository:** [https://github.com/anikdey72196-ops/Unit-converter.git]

A modern, fast, and beautifully designed single-page Unit Converter built with Django and Vanilla JavaScript.

## Features
- **Dynamic Conversions:** Instant calculations happening on the client-side without page reloads.
- **Multiple Categories:** Convert between units of Length, Weight, and Temperature.
- **Premium UI/UX:** Clean aesthetics featuring glassmorphism, responsive design, and smooth animations.

## Getting Started

1. **Clone the repository**
2. **Activate your virtual environment** (optional but recommended)
3. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the development server:**
   ```bash
   python manage.py runserver
   ```
5. **Open in browser:** Navigate to `http://127.0.0.1:8000`
